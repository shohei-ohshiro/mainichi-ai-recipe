先月、取引先からの長い提案書をChatGPTに一発で「要約して」と投げたら、肝心の金額の桁を丸められて、そのまま社内共有しかけたことがあります。私も先月までそうでした。

この記事を読むと、**長い資料をAIに1回で丸投げせず、3段階に分けて聞くことで、数字の見落としを防ぎながら要点を掴める**ようになります。

## なぜ「一発要約」だと数字が消えるのか

AIは長い文章の「真ん中あたり」の情報を軽く扱いやすいという性質があります。資料の最初と最後は拾えても、途中に出てくる金額や日付を読み落としたり、丸めて書いてしまうことがあります。1回のお願いに「全体像もつかみたいし、数字も知りたいし、リスクも心配」と全部詰め込むほど、この見落としは起きやすくなります。

## レシピ本体：3段階に分けて聞く

やることは、質問を1回でなく3回に分けるだけです。合計の所要時間は5分ほどです。

**ステップ1: まず全体構造だけをつかむ（1分)**

```
以下の資料を読んで、
1. この資料の結論を1行
2. 章立て（見出し）を箇条書きで
3. 誰向けの資料かを1行
だけ答えてください。数字の細部はまだ聞きません。

資料:
（ここに長い資料やPDFの中身を貼る）
```

**ステップ2: 数字だけを抜き出す（2分)**

```
同じ資料から、意思決定に関わる数字（金額・日付・件数など）だけを、
資料に書いてある通りに引用してください。
丸めたり、概算に直したりしないでください。
どの見出しの下にあった数字かも添えてください。
```

**ステップ3: リスク・懸念点だけを聞く（2分)**

```
同じ資料から、判断を誤ると困る懸念点やリスクを最大3つ、
それぞれ「何が起きうるか」を1行で挙げてください。
資料に書いていないことは推測で補わず「記載なし」としてください。
```

3つの答えを並べて読むだけで、資料全体を読むより先に「これは自分で原本を確認すべき箇所」が見えてきます。

## つまずきポイント

**その1: ChatGPT無料版は1日に扱えるファイル数に上限がある。** 無料プランは1日あたり数ファイルまでという制限があるため、毎日たくさんの資料を扱うなら有料プランか、Gemini・NotebookLMなど他のツールに振り分ける必要があります。

**その2(効かなかったこと): 契約書や金額が絡む重要書類は、AI要約だけで判断しない方がいい。** ステップ2で数字を丸めない指示を入れても、AIの読み取りミスがゼロになるわけではありません。金額や期日など「間違えたら取り返しがつかない」情報は、AIの回答をきっかけに原本の該当箇所を自分の目で確認する、という一手間を必ず挟んでください。一発要約をそのまま信じて共有する使い方は、この手の資料では逆に手間を増やします。

## まとめ：今日はこれだけ

今、開いていないまま溜まっている一番長い資料を1つ選んで、ステップ1のプロンプトだけ試してください。全体像をつかむのに1分もかかりません。
